## **2026-05-07**

**合约 V3 及测试网 — 新增修改订单接口**

- 在合约 V3 及合约测试网（中英文）中新增 `PUT /fapi/v3/order` — **修改订单 (TRADE)** 接口，位置在「批量下单」章节之前：
  - 支持修改已有 `LIMIT` 订单的 `quantity` 与 `price`
  - **请求中 `quantity` 与 `price` 均必须发送**
  - `orderId` 与 `origClientOrderId` 必须至少发送一个（同时发送则以 `orderId` 为准）
  - 新订单若不满足 `PRICE_FILTER` / `PERCENT_FILTER` / `LOT_SIZE` 限制，修改会被拒绝且原订单保留
  - 同一订单修改次数最多 10000 次

**合约 V3 及测试网 — 子账户划转规则更新**

- 在合约 V3 及合约测试网（中英文）中更新 `子账户划转` 接口文档：
  - **划转场景说明**表中新增一行：`子账户 A → 子账户 B` 使用**子账户钱包私钥**签名（此时来源地址即为签名子账户，无需传入 `fromAccountAddress`）
  - **注意事项**更新：使用**子账户私钥**签名时，既支持向**母账户**划转，也支持**子→子**划转（此前子→子必须由母账户签名）
