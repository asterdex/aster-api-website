## **2026-04-03**

**充值提现 API 更新**

- 新增两个 V3 版本查询接口（使用 Pro API-KEY 签名 V3）：
  - `POST /fapi/v3/aster/user-withdraw-info` — 查询用户提现限额及各链余额信息
  - `POST /fapi/v3/aster/deposit-withdraw-history` — 查询充值提现历史记录
- 以下 V1 接口已废弃，将在未来停止服务：
  - `POST /fapi/aster/user-withdraw-info`
  - `POST /fapi/aster/deposit-withdraw-history`
  - `POST /fapi/aster/user-withdraw`（EVM，合约）
  - `POST /sapi/api/v1/aster/user-withdraw`（EVM，现货）
  - `POST /fapi/aster/user-solana-withdraw`（Solana，合约）
  - `POST /sapi/api/v1/aster/user-solana-withdraw`（Solana，现货）
