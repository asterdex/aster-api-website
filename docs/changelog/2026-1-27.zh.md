**Aster Code 相关接口上线**

**Futures V3 API 更新**

- Market Data 新增 **查询资金费率配置** 接口 (`GET /fapi/v3/fundingInfo`)
- **期货现货互转** 安全类型标签由 `(USER_DATA)` 更新为 `(TRANSFER)`
- 接口标签更新：`POST /fapi/v3/asset/wallet/transfer` 由 `(HMAC SHA256)` 改为 `(TRANSFER)`
- 中文文档新增 **期货现货互转 (TRANSFER)** 章节

**Futures API 更新**

- Account & Trades 新增 **剩余可开仓名义价值** 接口 (`GET /fapi/v1/remainingOpenableNotionalValue`)
- **期货现货互转** 安全类型标签由 `(TRADE)` 更新为 `(TRANSFER)`
