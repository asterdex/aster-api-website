## **2026-05-12**

**Futures V3 与测试网 — 注册并授权 Agent 接口**

- 在 Futures V3 与 Futures Testnet 新增接口 `POST /fapi/v3/registerAndApproveAgent` —— **注册并授权 Agent (PUBLIC)**（EN & ZH）：
  - 一次调用完成 API Agent 账户注册与交易/提现权限授予。
  - **无需鉴权** —— 通过请求参数中的 `signature` 验证，无需 API Key 或 HMAC 请求头。
  - `signature` 通过对标准消息体（`msg: user={user}&nonce={nonce}&agentName={agentName}&agentAddress={agentAddress}&expired={expired}&signatureChainId={signatureChainId}&canSpotTrade={canSpotTrade}&canPerpTrade={canPerpTrade}&canWithdraw={canWithdraw}&ipWhitelist={ipWhitelist}`）使用用户钱包私钥签名生成。
  - 支持 **EVM**（EIP-712 Typed Data，`signatureChainId=56`）和 **Solana**（Ed25519，`signatureChainId=101`）。EVM 签名时需将上述 `msg` 字符串作为 `message.msg` 套入文档中的 EIP-712 typed-data 结构。
  - EIP-712 domain `chainId`：V3 主网为 **1666**，测试网为 **714**。
  - 可配置 Agent 权限：`canSpotTrade`、`canPerpTrade`、`canWithdraw`。
  - `ipWhitelist`：空格分隔，支持 CIDR；**`canWithdraw=true` 时必填且不可为空**。
  - 可选邀请码 `agentCode`；Agent 有效期由 `expired`（毫秒时间戳）控制。
  - 权重：**50**。
