## **2026-05-26**

**Prediction Testnet（预测市场测试网） — 新增文档章节**

- 新增文档章节 **Prediction Testnet**，覆盖测试网的预测市场 API（EN & ZH）。
- 参照 `futures-v3` 文件夹的拆分约定，将文档拆分为 `docs/prediction-testnet/` 下 6 个本地文件：
  - `general-info.md` —— 概览、Agent 设置、访问限制、鉴权方式、签名示例、公共 API 参数、过滤器
  - `market-data.md` —— 行情接口（`exchangeInfo`、`depth`、`klines`、`ticker` 等）
  - `account&trades.md` —— 现货账户与交易，以及预测市场铸造 / 销毁 / 持仓 / 结算 / 提现接口
  - `websocket-market-streams.md` —— WebSocket 行情推送
  - `user-data-streams.md` —— WebSocket 账户信息推送（listen key、账户更新、订单更新等）
  - `error-codes.md` —— 错误码参考（10xx / 11xx / 20xx / 40xx）
- 主要内容：
  - **REST Base URL**：`https://papi.asterdex-testnet.com`
  - **WebSocket Base URL**：`wss://pstream.asterdex-testnet.com`
  - 独立的 exchange-info 接口：`GET /api/v3/prediction/exchangeInfo`
  - 预测市场 K 线间隔简化为仅 `1s` 与 `1m`
